# Aeon Trespass Expert

IR-first document compiler for the Aeon Trespass board game rulebook.
Translates the English rulebook into Russian with first-class icon/symbol handling,
deterministic artifacts, and a static React reader.

## Quick start

```bash
# Python workspace
uv sync

# Node workspace
pnpm install

# Run all tests
make test

# Lint everything
make lint
```
